About This Project

This application uses a machine learning model to predict the likelihood of a hotel booking being canceled. By analyzing various factors such as lead time, booking changes, deposit type, and guest history, the model provides insights that can help hotels manage reservations and reduce revenue loss from cancellations.

API Access

This prediction service is also available via a REST API endpoint. Send a POST request to /api/predict with JSON data containing the booking information.
